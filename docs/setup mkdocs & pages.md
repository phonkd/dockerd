## Setup mkdocs

1. Install python and pip `sudo pacman -S python python-pip`
2. Install mkdocs (pip) `pip install mkdocs-material`
3. Initialize project `mkdocs new .`
4. Edit mkdocs.yml (e.g add theme:)
5. mkdocs serve
