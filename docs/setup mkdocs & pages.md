   ## Setup mkdocs
   1. Install python and pip `sudo pacman -S python python-pip`
   2. Install mkdocs (pip) `pip install mkdocs mkdocs-material pymdown-extensions markdown`
   3. Create `mkdocs.yml` file in root repository `nvim mkdocs.yml`
   4. Edit mkdocs.yml
